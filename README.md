# 30 Days of Python

This repository is a beginner-friendly Python learning project based on a 30-day practice journey. It contains simple scripts, explanations, and examples that make core Python concepts easier to learn.

## What is inside

### Day 1
- `hello_word.py` - basic print statements and arithmetic examples
- `basics.py` - Python built-in functions such as `len()`, `round()`, `abs()`, `sum()`, `min()`, `max()`, `type()`, `bool()`, `range()`, `id()`, and `dir()`
- `variables.py` - variable basics, arithmetic with variables, and simple personal data examples
- `data_type.py` - common Python data types such as numbers, strings, lists, tuples, sets, and dictionaries

### Day 2
- `arthmetic_operators.py` - arithmetic operators with clearer examples
- `comparison_oprators.py` - comparison operators with labeled output
- `logcal_operators.py` - logical operators with explanations
- `assignment_operator.py` - assignment and compound assignment operators
- `string.py` - string methods, slicing, joining, replacing, splitting, and checks
- `exercise_day2.py` - Day 2 exercise solutions in Python
- `exercise_day2.md` - formatted learning guide with questions and answers

## What we have achieved so far

## Goal

The goal of this repository is to make Python easier to learn through small, clear, practice-based examples.

## Run the scripts

You can run any file with Python, for example:

```bash
python Day1/hello_word.py

```
